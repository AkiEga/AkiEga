### Tools I've Built

#### VS Code Extensions
- [file-open-history](https://github.com/AkiEga/file-open-history) — Tracks and displays a history of file-opening actions in VS Code via a text editor panel or output channel
- [vscode-extension-ascii-sketch](https://github.com/AkiEga/vscode-extension-ascii-sketch) — A VS Code extension for creating and editing ASCII art in `.ascii` files with canvas generation, smart box drawing, and box alignment tools
- [vscode-extension-section-info](https://github.com/AkiEga/vscode-extension-section-info) — A VS Code extension that copies selected code sections with contextual metadata (file path, line, git info) to the clipboard in customizable formats
- [vscode-extension-svg-sketch](https://github.com/AkiEga/vscode-extension-svg-sketch) — A VS Code extension for creating and editing SVG diagrams with a canvas-based editor, shapes, arrows, and clipboard image support
- [vscode-extension-tree](https://github.com/AkiEga/vscode-extension-tree) — A VS Code extension that runs the `tree` command on any folder via the Explorer sidebar and displays the output in an editor or webview
- [vscode-msvs-proj-manager](https://github.com/AkiEga/vscode-msvs-proj-manager) — A VS Code extension that parses Microsoft Visual Studio solution files and opens terminals in project root directories on Windows
#### Embedded Systems / Hardware
- [pico_arcade_controller](https://github.com/AkiEga/pico_arcade_controller) — Arcade joystick controller built with Raspberry Pi Pico (USB HID). Supports mouse cursor movement, long-press acceleration, and window switching via rapid left/right inputs.
- [ega-right-kb](https://github.com/AkiEga/ega-right-kb) — Custom right-side keyboard powered by Raspberry Pi Pico with GPIO key mapping and HID support.
