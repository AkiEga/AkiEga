### Tools I've Built

#### VS Code Extensions
- [file-open-history](https://github.com/AkiEga/file-open-history) — Tracks and shows your VS Code file open history in an editor pane or output channel
- [vscode-extension-ascii-sketch](https://github.com/AkiEga/vscode-extension-ascii-sketch) — A VS Code extension for creating and editing ASCII art with canvas, box, and alignment tools
- [vscode-extension-section-info](https://github.com/AkiEga/vscode-extension-section-info) — Copies selected code section details and Git context to the clipboard in customizable formats
- [vscode-extension-svg-sketch](https://github.com/AkiEga/vscode-extension-svg-sketch) — A VS Code extension for creating and editing SVG diagrams with a canvas-based editor
- [vscode-extension-tree](https://github.com/AkiEga/vscode-extension-tree) — Runs the tree command on folders in VS Code and shows the file tree in an editor or webview
- [vscode-msvs-proj-manager](https://github.com/AkiEga/vscode-msvs-proj-manager) — VS Code extension that parses Visual Studio solution files and opens terminals in project root folders
#### Embedded Systems / Hardware
- [pico_arcade_controller](https://github.com/AkiEga/pico_arcade_controller) — Arcade joystick controller built with Raspberry Pi Pico (USB HID). Supports mouse cursor movement, long-press acceleration, and window switching via rapid left/right inputs.
- [ega-right-kb](https://github.com/AkiEga/ega-right-kb) — Custom right-side keyboard powered by Raspberry Pi Pico with GPIO key mapping and HID support.
