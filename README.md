### Tools I've Built

#### VS Code Extensions
- [file-open-history](https://github.com/AkiEga/file-open-history) — Tracks and shows your file opening history in VS Code in an editor pane or output channel
- [vscode-extension-ascii-sketch](https://github.com/AkiEga/vscode-extension-ascii-sketch) — A VS Code extension for creating and editing ASCII art with canvas, box drawing, and box alignment tools
- [vscode-extension-section-info](https://github.com/AkiEga/vscode-extension-section-info) — Copies selected code section details and customizable file, line, and Git info to the clipboard
- [vscode-extension-svg-sketch](https://github.com/AkiEga/vscode-extension-svg-sketch) — A VS Code extension for creating and editing SVG diagrams in a canvas-based editor with Vim-style modes
- [vscode-extension-tree](https://github.com/AkiEga/vscode-extension-tree) — Run the tree command on folders in VS Code and view the file structure in an editor or webview
- [vscode-msvs-proj-manager](https://github.com/AkiEga/vscode-msvs-proj-manager) — Manage Visual Studio solution files in VS Code by parsing .sln files and opening project-root terminals
#### Embedded Systems / Hardware
- [pico_arcade_controller](https://github.com/AkiEga/pico_arcade_controller) — Arcade joystick controller built with Raspberry Pi Pico (USB HID). Supports mouse cursor movement, long-press acceleration, and window switching via rapid left/right inputs.
- [ega-right-kb](https://github.com/AkiEga/ega-right-kb) — Custom right-side keyboard powered by Raspberry Pi Pico with GPIO key mapping and HID support.
