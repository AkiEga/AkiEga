### Tools I've Built

#### VS Code Extensions
- [file-open-history](https://github.com/AkiEga/file-open-history) — A VS Code extension that tracks and displays your file-opening history in a text editor or output channel
- [vscode-extension-ascii-sketch](https://github.com/AkiEga/vscode-extension-ascii-sketch) — A VS Code extension for creating and editing ASCII art with canvas generation, smart box drawing, and dedicated `.ascii` file support
- [vscode-extension-section-info](https://github.com/AkiEga/vscode-extension-section-info) — Copies selected code sections to clipboard with customizable metadata like file path, line numbers, git info, and language
- [vscode-extension-svg-sketch](https://github.com/AkiEga/vscode-extension-svg-sketch) — A VS Code extension for creating and editing SVG diagrams using a canvas-based editor with shapes, arrows, text, and templates
- [vscode-extension-tree](https://github.com/AkiEga/vscode-extension-tree) — A VS Code extension that runs the `tree` command on folders from the Explorer sidebar and displays the output in an editor or webview
- [vscode-msvs-proj-manager](https://github.com/AkiEga/vscode-msvs-proj-manager) — A VS Code extension that parses Microsoft Visual Studio solution files and opens terminals in project root directories
#### Embedded Systems / Hardware
- [pico_arcade_controller](https://github.com/AkiEga/pico_arcade_controller) — Arcade joystick controller built with Raspberry Pi Pico (USB HID). Supports mouse cursor movement, long-press acceleration, and window switching via rapid left/right inputs.
- [ega-right-kb](https://github.com/AkiEga/ega-right-kb) — Custom right-side keyboard powered by Raspberry Pi Pico with GPIO key mapping and HID support.
